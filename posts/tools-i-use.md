I change my setup less than I used to. Here's what's stuck.

## Editor

VS Code with a minimal set of extensions. I've tried Neovim twice. I always come back.

## Terminal

Ghostty. It's fast, it looks good, and it doesn't try to be clever.

## Notes

Plain markdown files in a folder. No app, no sync service. Just files.

## Version control

Git, obviously. I've made peace with the fact that I'll never fully understand the reflog.
